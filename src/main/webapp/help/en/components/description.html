Description

The description represents a quick presentation of the content of the page. This descriptpion is used for referencing both internally and internally. It is also used in search results or in components that refer to the page. It is possible to use the component only in the context of referencing and not to display it in the content of the page, for that it is enough to put it 'hidden' via the tab "settings".

In order to create a default description it is possible to place a description on a parent page and repeat it, it will then be used on the child pages unless a description component is already there. It is possible to customize these descriptions with variables. List of main variables:

  • ${info.pageName} : name of the page.
  • ${info.pageTitle} : title of the page.
  • ${info.sectionTitle} : title of section (second level of navigation).
  • ${info.subSectionTitle} : title of sub-section (third level of navigation).
  • ${info.publishDate} : latest date of publication.
  • ${info.date} : date of the page. (short display)
  • ${info.mediumDate} : date of the page. (current date)
  • ${info.fullDate} : date of the page. (current date)
  • ${info.nowDate} : current date. (short display)
  • ${info.nowMediumDate} : current date. (current date)
  • ${info.nowDate} : current date. (current date)
  • ${info.currentDay} : current day
  • ${info.currentMonth} : current month
  • ${info.currentYear} : current year

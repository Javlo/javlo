Description

La description représente une présentation rapide du contenu de la page. Cette descritpion est utilisées pour le référencement autant externne que internne. Elle est également utilisé dans les résultats de recherche ou dans les composants qui font références à la page. Il est possible de n'utiliser le composant que dans le cadre du référencement et ne pas l'afficher dans le contenu de la page, pour cela il suffit de le mettre 'hidden' via l'onglet "settings".

Afin de pouvoir créer une description par défault il est possible de placer une description sur une page parente et de la répéter, elle sera alors utilisé sur les pages filles sauf si un composant description s'y trouve déjà. Il est possible de personnaliser ces descriptions avec des variables. Liste des variables principales :

  • ${info.pageName} : nom de la page.
  • ${info.pageTitle} : titre de la page.
  • ${info.sectionTitle} : titre de la section (deuxième niveau de navigation).
  • ${info.subSectionTitle} : titre de la sous section (troisième niveau de navigation).
  • ${info.publishDate} : dernière date de publication de la page.
  • ${info.date} : date du contenu de la page. (affichage court)
  • ${info.mediumDate} : date du contenu de la page. (affichage moyen)
  • ${info.fullDate} : date du contenu de la page. (affichage long)
  • ${info.nowDate} : date courante. (affichage court)
  • ${info.nowMediumDate} : date courante. (affichage moyen)
  • ${info.nowDate} : date courante. (affichage long)
  • ${info.currentDay} : jour courant
  • ${info.currentMonth} : mois courant
  • ${info.currentYear} : année courante
